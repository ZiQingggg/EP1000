Anime Moodlight Box

A Anime moodlight box delicately handcrafted with the finest ply wood and slick acrylic .Features include music and a touch sensor as the switch to play the light and music (Please take note the touch sensor isn't working as I was unable to complete it in time and that I had some trouble with the complexity of the coding due to the fact that I lack the knowledge to do it)

For this project I started out by creating sketch of my moodlight box

Next here are some of the Bill of Materials I will need in my moodlight box Neopixel ,8ohm Speakers ,Touch sensor ,Arduino UNO ,Male to Female Wires ,2n222 Transistor ,3mm Ply Wood ,3mm Transparent Acrylic ,220 resistors ,330 resistors ,Touch sensor ,Micro SD card Adaptor ,Micro SD card ,BreadBoard

Next here is a rundown of how I plan to complete this whereby first I will start by generating the code out,build a prototype,fusion 360 a model,inkscaping graphics for the acrylic ,Lasercut using epilog pro, 3D print out object and finally showcase the final product.

So first I basically started by programming the code for the lighting and Color of the lights whereby I created it by looking at some helpful tutorials and downloaded some adafruit library codes and added colors like blue,purple,pink,red,green and yellow next because I wanted the light to fade in ouy I then had to learn some python and asked for some advice from a friend who is used to doing this kind of stuff which he told me is hard but still achieveable.

After that I then started soldering my 8 ohm speakers and moved onto coding the code for making the music with these helpful tutorials as well as changed a bit of the layout of the wires

After that I then moved onto integrating the touch sensor as a switch but failed because of the complex programming of the fading in and fading out of the led pixels it affected touch sensor while also at the same time I also had some time constraints for my other pproject and not having much understanding about the codes .

Next moving on to the prototype I decided referenced from the older students works like these 2 student websites https://jyaprh.github.io/EP1000/final.html and https://nurin-n.github.io/EP1000/projectpage.html.
